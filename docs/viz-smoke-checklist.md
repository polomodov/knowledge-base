# Ручная smoke-проверка v4-визуализации

Проверка выполняется на реальном артефакте
<code>data/generated/viz/knowledge-base.html</code>, собранном командой
<code>kb viz build</code>. Артефакт содержит заголовки, URL и топологию личной
базы знаний: не публикуйте и не прикладывайте его к PR.

## Подготовка

- [ ] Перестроить необходимые derived-индексы и выполнить <code>kb viz build</code>.
- [ ] Убедиться, что команда завершилась успешно и сообщила путь и размер HTML.
- [ ] Запустить <code>node scripts/check-viz-template.mjs</code>.
- [ ] Отключить сеть или открыть DevTools → Network в режиме Offline.
- [ ] Открыть HTML напрямую через <code>file://</code>, без локального HTTP-сервера.
- [ ] Повторить основные сценарии в актуальных Chrome, Firefox и Safari.

## Graph export

- [ ] Собрать node-link JSON и GraphML командами из `docs/visualization.md`.
- [ ] JSON парсится, а `nodes`/`links` совпадают с числами в CLI-отчёте; document nodes не содержат body/chunk text.
- [ ] GraphML импортируется без ошибок в актуальный Gephi; видны кириллица, emoji, document/topic node kinds и edge kinds/weights.
- [ ] Тот же GraphML открывается в yEd, typed `weight`, `chunk_pairs` и `document_count` читаются как ожидаемые типы.
- [ ] `--ego DOCUMENT_KEY` помечает центр через graph-level `ego_document_key` и node `is_ego_center=true`.
- [ ] JSON и GraphML одного запуска совпадают по числу узлов/рёбер и ключам связей.

## Общие сценарии

- [ ] Страница открывается без внешних сетевых запросов и ошибок в Console.
- [ ] Кириллица, emoji и специальные символы в заголовках отображаются без повреждений.
- [ ] Табы «Карта», «Таймлайн» и «Документ» переключаются мышью и клавиатурой.
- [ ] Поиск находит документы по заголовку и связанным топикам; Escape закрывает результаты.
- [ ] Выбор результата открывает ego-вид нужного документа.
- [ ] В футере видны время сборки, база, модель embeddings, счётчики и доступные index runs.
- [ ] Обычная сборка показывает «Черновики включены: нет» и не содержит безопасный тестовый draft.
- [ ] Сборка с <code>--include-drafts</code> показывает «Черновики включены: да» и содержит этот тестовый draft.
- [ ] Провенанс-ссылка открывает только <code>http://</code> или <code>https://</code> URL в новой вкладке.

## Карта сообществ и топиков

- [ ] Сообщества, топики и точки документов видны после открытия страницы.
- [ ] Легенда источников объясняет цвета точек документов.
- [ ] Точки документов появляются пакетами и интерфейс не зависает на полном корпусе.
- [ ] Ползунок порога меняет число видимых co-occurrence-рёбер; начальное значение — 5.
- [ ] Колесо/трекпад масштабирует карту относительно указателя.
- [ ] Перетаскивание панорамирует карту, «Сбросить масштаб» возвращает исходный вид.
- [ ] Наведение на точку показывает заголовок, источник и дату.
- [ ] Клик по точке открывает ego-вид выбранного документа с допустимой provenance-ссылкой.
- [ ] Изолированные документы отображаются в зоне <code>unclustered</code>, если они есть в payload.

## Таймлайн

- [ ] Режим «Источники» показывает stacked-бары по месяцам и легенду источников.
- [ ] Режим «Топики» показывает линии top-topic series и соответствующую легенду.
- [ ] Подписи месяцев остаются читаемыми на узком и широком окне.
- [ ] Наведение на бары и линии показывает серию, месяц и значение там, где применимо.
- [ ] Колесо масштабирует таймлайн, перетаскивание панорамирует, кнопка сброса возвращает исходный вид.
- [ ] Изменение размера окна не обрезает график и не вызывает ошибок.

## Ego-граф документа

- [ ] В центре находится выбранный документ, в первом кольце — не более 10 соседей.
- [ ] Второе кольцо ограничивает весь вид примерно 60 узлами и не блокирует интерфейс.
- [ ] Повторный клик по соседу перестраивает ego-граф вокруг него детерминированно.
- [ ] Колесо масштабирует ego-граф, перетаскивание фона панорамирует, кнопка сброса возвращает исходный вид.
- [ ] В боковой панели показаны источник, дата, ключ, сообщество, топики и provenance-ссылка.
- [ ] Видна честная подпись: «Показаны топ-10 соседей; полный граф — <code>kb export graph</code>».
- [ ] При отсутствии similarity-рёбер показано понятное предупреждение, страница не падает.

## Вырожденные состояния

- [ ] HTML, собранный из пустой БД, открывается и показывает «Нет данных», без ошибок Console.
- [ ] Payload без communities/topics всё ещё позволяет открыть таймлайн и карточку документа.
- [ ] Недопустимый URL (<code>javascript:</code>, <code>data:</code>, <code>file:</code>) не становится ссылкой.
- [ ] Длинный заголовок с кириллицей, emoji, кавычками, амперсандом и закрывающим script-тегом отображается как текст.
